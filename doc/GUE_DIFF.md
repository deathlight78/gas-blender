# GUE Decoplanner vs Gas Blender — 감압 계획 비교 분석

> 분석 기준: `reference/decoplanner-offline.html` (GUE Decoplanner 기반 웹앱)
> 분석일: 2026-04-05

---

## 공통점

- Bühlmann ZHL-16C 기반 감압 알고리즘
- GF Low / High 지원
- Multi-gas (감압 기체 전환) 지원
- CNS% / OTU 산소 독성 계산
- 3m (미터계) / 10ft (야드계) 단위 감압 정지
- 메트릭 / 임페리얼 단위 지원

---

## 기능 비교

| 항목 | Gas Blender (현재) | GUE Decoplanner |
|---|---|---|
| **알고리즘** | ZHL-16C 단일 | ZHL-16B/C + **VPM 동시 계산** |
| **ZHL 기본 계수** | ZHL-16C | **ZHL-16B** (GUE 표준), C 선택 가능 |
| **GF 기본값** | 설정에서 지정 | **GF Low 20% / High 85%** (GUE 표준) |
| **Bailout GF** | 없음 | **GF Low 30% / High 90%** |
| **CCR / pSCR** | 없음 | **완전 지원** (setpoint 기반, pSCR 희석비 설정) |
| **표면 간격** | 없음 | **지원** (조직 포화도 반영) |
| **반복 다이빙** | 없음 | **지원** |
| **고도 보정** | 없음 | **최대 3,000m 고도** |
| **RMV/가스량 연동** | 별도 탭 단순 계산 | **프로파일 연동** — 정지별 실제 소비량 자동 계산 |
| **마지막 정지** | 3m 고정 | **3m 또는 6m 선택** 가능 |
| **ICD 경고** | 없음 | **역방향 가스 전환(N₂↑) 경고** |
| **최소 ppO₂ 경고** | 없음 | **0.16 bar 미만 경고** |
| **심층 GF 멀티레벨** | 없음 | **GF Multilevel Mode** |
| **정지 시간 배분** | 단일 계산 | model / equal / linear / s 4가지 방식 |
| **보존성 레벨** | GF로 간접 조절 | **conservatism 레벨** 별도 제공 |

## Gas Blender의 장점

- **모바일 앱** — 오프라인 사용, 화면 전환 불필요
- **직관적 UI** — 드럼롤 피커 + 슬라이더로 빠른 입력
- **통합 워크플로** — 블렌딩 → 가스 계획 → 감압 계획을 한 앱에서
- **실린더 계획 탭** — SAC / 가스량 / 실린더별 사용시간 통합 관리
- **코드 단순성** — 유지보수 용이, 커스텀 쉬움

## 개선 우선순위 (실용성 기준)

### ✅ Priority 1 — RMV 연동 (가스 소비량 자동 계산)
- 감압 계획 입력에 바닥 RMV / 감압 RMV 추가
- 각 세그먼트 및 정지별 가스 소비량(L) 자동 계산
- 기체별 총 소비량을 결과 화면에 표시
- **효과**: 실린더 계획 탭과 연결되어 실전 다이빙 준비에 직결

### ✅ Priority 2 — ICD 경고 (역방향 가스 전환 경고)
- 감압 기체 전환 시 신규 기체의 fN₂ > 이전 기체의 fN₂ 이면 경고
- 예: TMX 21/35 → EAN50 전환 시 N₂ 증가 → ICD 위험
- **효과**: 안전 체크 강화, 기포 형성 위험 사전 감지

### ✅ Priority 3 — 마지막 정지 수심 사용자 설정
- 기본값 **6m** (GUE 표준)
- 사용자가 3m ~ 9m 범위에서 조절 가능
- **효과**: GUE 다이버 및 보수적 플래닝 선호 사용자 대응

---

## 미구현 (장기 검토)

- VPM (Varying Permeability Model) 알고리즘 — 구현 복잡도 높음
- CCR / pSCR 감압 계획 — 별도 setpoint 로직 필요
- 표면 간격 / 반복 다이빙 — 상태 영속성 설계 필요
- 고도 보정 — 해수면 압력 파라미터화로 상대적으로 간단
- ZHL-16B 계수 선택 — 계수 테이블 추가로 구현 가능
